- [ ] **The Context API & Prop Drilling**  
    - MovieContext.tsx and "Prop Drilling" are key concepts.  
    - Prop Drilling: Passing data through many layers as props.  
    - Context API: Creates a global data store accessible via useContext.

- [ ] **React Router (Navigation)**  
    - Using BrowserRouter, Routes, and useNavigate for SPA navigation.  
    - BrowserRouter: Enables navigation.  
    - Routes & Route: Maps URLs to components.  
    - Link: React-safe navigation without page refresh.  
    - useNavigate: Programmatic navigation.

- [ ] **Controlled Components**  
    - Example: `<input value={name} onChange={(e) => setName(e.target.value)} />`  
    - Controlled Input: State is the source of truth, not the DOM.  
    - Enables features like debouncing.

- [ ] **The Testing Stack (Vitest & Testing Library)**  
    - Tests like Button.test.tsx are present.  
    - Vitest: Test runner.  
    - React Testing Library: Tools for rendering and interacting with components in tests.  
    - Purpose: Automate testing to avoid manual checks after code changes.

- [ ] **Hooks (useState, useEffect, useRef, useMemo)**  
    - useState: Manage local state.  
    - useEffect: Side effects like data fetching.  
    - useRef: Access DOM nodes or persist values.  
    - useMemo: Optimize performance by memoizing values.

- [ ] **Tailwind CSS Basics**  
    - Utility-first CSS framework.  
    - Compose styles directly in JSX.  
    - Responsive design with classes like `md:` and `lg:`.

- [ ] **Component Composition & Reusability**  
    - Build reusable UI blocks.  
    - Props and children for flexible components.

- [ ] **Error Handling in React**  
    - Try/catch in async functions.  
    - Error boundaries for catching render errors.

- [ ] **Fetching Data (Async/Await, Fetch API, Axios)**  
    - Use async/await for asynchronous code.  
    - Fetch API or Axios for HTTP requests.
